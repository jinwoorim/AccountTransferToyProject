# AccountTransferToyProject

 - SpringFramework와 MyBatis와 JSP를 학습하여 진행한 프로젝트
 - 우리은행과 국민은행 간의 계좌 이체 프로그램

## 기술 스텍

 - Language : Java17
 - Framework : Spring(v5.3.9), Mybatis(v3.3.1)
 - View : JSP
 - WAS : Tomcat(v9.0)
 - DB : H2(2.1.212)
